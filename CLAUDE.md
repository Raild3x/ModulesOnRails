## Agent skills

### Issue tracker

Issues live in GitHub Issues on Raild3x/ModulesOnRails (via the `gh` CLI);
external PRs are not a triage surface. See `.github/agents/issue-tracker.md`.

### Triage labels

Default vocabulary: needs-triage, needs-info, ready-for-agent,
ready-for-human, wontfix. See `.github/agents/triage-labels.md`.

### Domain docs

Multi-context: `CONTEXT-MAP.md` at the root points to per-package
`CONTEXT.md` files under `lib/<package>/`. See `.github/agents/domain.md`.
